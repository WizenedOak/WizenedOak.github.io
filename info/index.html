    Thank you for taking the time to look at my website. Alot of time went into the designing, coding, and content creating for it, with me going through multiple iterations of it all. So it means a lot to me that you took the time to explore and appreciate it.     Tools Used

    This website was made using HTML, Nunjucks, and some JavaScript for functionality and components, CSS for styling, 11ty for static site generation, and github actions for building and deploying the site.
